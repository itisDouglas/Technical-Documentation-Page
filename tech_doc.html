Introduction to Python

Python is simple to use, but it is a real programming language, offering much more structure and support for large programs than shell scripts or batch files can offer. On the other hand, Python also offers much more error checking than C, and, being a very-high-level language, it has high-level data types built in, such as flexible arrays and dictionaries. Because of its more general data types Python is applicable to a much larger problem domain than Awk or even Perl, yet many things are at least as easy in Python as in those languages.

Python allows you to split your program into modules that can be reused in other Python programs. It comes with a large collection of standard modules that you can use as the basis of your programs — or as examples to start learning to program in Python. Some of these modules provide things like file I/O, system calls, sockets, and even interfaces to graphical user interface toolkits like Tk.

Python is an interpreted language, which can save you considerable time during program development because no compilation and linking is necessary. The interpreter can be used interactively, which makes it easy to experiment with features of the language, to write throw-away programs, or to test functions during bottom-up program development. It is also a handy desk calculator.

By the way, the language is named after the BBC show “Monty Python’s Flying Circus” and has nothing to do with reptiles. Making references to Monty Python skits in documentation is not only allowed, it is encouraged!

Hello World!

To see what can be done with Python, or any programming language for that matter, printing out the words "Hello World!" is a tradition for most programmers venturing to new languages. Fortunately Python allows this to be done quickly. See the below code.

>>> print("Hello World!")
Hello World!
>>>

It is assumed the reader of this documentation has the latest version of Python installed and ready to go.

Variables

Python is a simple language. Notice that there are no semicolons required. Instead, Python depends on spacing and indentation.

The next few examples demonstrate how to declare variables.

>>> x = 4
>>> y = 7
>>> x + y
11

In Python a variables type doesn't need to be specified, the interpreter already infers the type.

Constants

There is no special way to declar constants in Python. There is only one rule and that is to capitalize your constant's name. See the below example:

>>> SALES_TAX = 7.5
>>> print(SALES_TAX)
7.5

Data Types

The following the six standard data types in Python:

  • Integer
  • Float
  • String
  • List
  • Tuple
  • Set
  • Dictionary

The following are examples of each of these data types:

>>> #Integer
>>> a = 5
>>> print(a)
5

>>> #Float
>>> b = 5.1
>>> print(b)
5.1

>>> #String
>>> word = "Hello!"
>>> print(word)
Hello!

>>> #List
>>> List = [2, 4, 5, "Hello again!"]
>>> print(List[3])
Hello again!

>>> #Tuple
>>> aTuple = (1, 2, 3, "Python")
>>> print(aTuple[3])
Python

>>> #Set
>>> aSet = {1,2,3,"More Python"}
>>> print(aSet)
{1, 2, 3, 'More Python'}

>>> #Dictionary
>>> Dict = {1: "hello", 2:"hello, again", 3: "Hi"}
>>> print(Dict) {1: 'hello', 2: 'hello, again', 3: 'Hi'}
>>> print(Dict[2]) hello, again

If...else Statement

In python the if statement is written with "if" keyword followed by the variables you want to evaluate and the logical conditions; you finish the if statement with a colon.

After the colon, you indent, letting the Python interpreter know that whatever comes after the colon is what should happen after the condition has been evaluated.

When you need an else statement, simply indent to the line below and use the keyword else followed by a colon. After the colon, simply write whatever will happen if the initial evaluation fails. See the below example:

>>> #If Else
>>> a = 11
>>> b = 100
>>> if b > a: print("b is greater than a") else: print("a is greater than b")
b is greater than a

While Statement

One of the loop commands that can be accomplished with Python is the while loop. Similar in declaration to the if statement, the while loop iterates through a set of statements. See the below example:

>>> #While Loop
>>> i = 1
>>> while i < 6: print(i) i += 1
1 2 3 4 5

Function Declarations

A function is a block of code which only runs when it's called.

You can pass data, known as parameters, into a function.

A function can return data as a result.

The following is an example on how you create and use a function.

>>> #Function
>>> def my_function(): print("hello from a function")
>>> #here, I am calling my function
>>> my_function()
hello from a function
